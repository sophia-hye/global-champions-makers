### B2B/제휴 탭 (필드 차이)

```
회사/브랜드명 ___ · 담당자 ___ · 연락처/이메일 ___
제휴 유형 [ 팝업 스폰서십 / 공간 대여 / 제품 브리핑 ▾ ]
제안 내용 [ 텍스트 영역 ]      [ 제휴 문의하기 ]
```

- 제출 → 성공 토스트 + 확인 이메일/Slack 알림
- 유효성 검증: 필수값, 이메일/전화 형식, 동의 체크

## 6. 반응형 브레이크포인트

| 구분 | 폭 | 레이아웃 |
|------|-----|----------|
| Mobile | < 768px | 1열 스택, 햄버거 메뉴, CTA 풀폭 |
| Tablet | 768–1024px | 2열 카드 |
| Desktop | > 1024px | 3열 카드, 풀블리드 히어로 |
