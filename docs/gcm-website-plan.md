## 1. 프로젝트 개요

### 1.1 브랜드 정의

| 항목 | 내용 |
|------|------|
| 브랜드명 | GCM (이큐어) |
| 법인 | 주식회사 GCM / 이큐어 교육성장연구소 (ETERNITY COMPANY) |
| 카테고리 | 차세대 웰니스 교육 (Wellness Education) |
| 한 줄 정의 | 입시 컨설팅과 멘탈 케어를 혁신하는 글로벌 하이브리드 허브 |
| 슬로건 | The Next Generation Wellness Education |
| 클로징 메시지 | Are you ready to CURE & GROW? |

### 1.2 핵심 가치 제안 (Value Proposition)

경쟁과 압박에 내몰린 엘리트 청소년들을 위한 "완벽한 플랜 B".
단 하나의 길(엘리트 스포츠 입시)만 보고 달려온 아이들이 실패했을 때
이들을 다시 일으켜 세우는 가장 강력하고 견고한 **사회적 안전망(Social Safety Net)** 을 제공한다.

- **이성(E)**: 10년 경력 북미 입학사정관의 데이터 기반 입학 로드맵
- **감성(Q)**: 깊은 심리 치유와 멘탈 케어
- 두 축을 결합한 압도적인 플랜 B 설계

### 1.3 비즈니스 본질

아이들이 실패를 딛고 글로벌 웰니스 리더로 자립할 수 있도록 돕는 것.
이것이 GCM가 창출하는 **사회적 기업의 본질적 가치**이며,
재단/투자자에게는 "미래 세대를 위한 가장 단단한 심리적 인프라"에 대한 투자다.

## 4. 페이지별 콘텐츠 구성

### 4.1 Home

피치덱의 스토리라인(Problem → Solution → Brand → Model → Space → Impact)을 스크롤 내러티브로 재구성.

1. **Hero**: 로고 + "The Next Generation Wellness Education" + 핵심 카피("완벽한 플랜 B") + CTA(상담 신청 / 프로그램 보기)
2. **The Problem**: 엘리트 청소년 번아웃 문제 제기 (경고/임팩트 톤, 레드 액센트)
3. **Our Solution**: 사회적 안전망 = 이성(데이터) + 감성(치유)
4. **Programs 요약**: Core / Peer Mentoring 카드 3종
5. **Space 미리보기**: Sober Bar & Tech Showroom 비주얼
6. **Community / Crew**: 선순환 구조 (실패 → 회복 → 멘토)
7. **Closing CTA**: "Are you ready to CURE & GROW?" + 상담 신청

### 4.2 About

- The Problem / Our Solution 상세
- Brand Identity 섹션:
  - 로고 = E(이성) + Q(감성)의 결합
  - 두 알파벳이 만나 무한대(∞)의 잠재력 + 흔들리지 않는 닻(Anchor)
  - 우믈라우트(ü) = 북유럽 감성, 프리미엄 웰니스 라이프스타일 브랜드의 세련미

### 4.3 Programs

| 프로그램 | 설명 | 수익 모델 |
|----------|------|-----------|
| Core: 입시 & 멘탈 | 북미 대학 스포츠/일반 전형 진학 로드맵 + 심리 치료 세션 | 프리미엄 컨설팅 비용 (안정적 캐시카우) |
| Peer Mentoring | 회복·진학에 성공한 선배가 후배를 이끄는 글로벌 피어 멘토 | 커뮤니티/멤버십 |

### 4.4 Space

컨셉: **"병원이나 학원이 아닌, 힙한 아지트"**
노출 콘크리트 + 메탈 소재 + 네온 조명, 마치 스트리트 패션 플래그십 스토어 같은 감각적 만족감과 소속감.

- **Sober Bar**: 무알콜 음료, 글루텐프리 스낵. 학부모/선수 전용 휴식 + 식음료 수익화
- **Tech Showroom**: 첨단 스포츠 과학·웰니스 멘탈케어 기기 팝업 전시 (B2B 브리핑 연계)
- 위치: SEOULITE HANNAM (테니스 코트 內 프리미엄 웰니스 공간)

### 4.5 Partners / B2B

- 하이엔드 스포츠 용품·웰니스 테크 브랜드의 오프라인 팝업 공간 대여
- 제품 브리핑 세션 + 고단가 B2B 광고/스폰서십
- 제휴 문의 폼

### 4.6 Impact / IR

- 사회적 기업 모델 설명, 선순환 구조 인포그래픽
- "미래 세대를 위한 심리적 인프라" 메시지
- 재단/투자자용 자료 다운로드(피치덱, 임팩트 리포트)

## 5. 디자인 컨셉 (Design System 방향)

피치덱 비주얼 톤 계승: **다크 모드 기반의 프리미엄 웰니스 + 스트리트 감성.**

### 5.1 컬러 팔레트 (제안)

| 역할 | 색상 | 용도 |
|------|------|------|
| Background | Deep Navy `#0F1B2D` 계열 | 기본 배경 |
| Primary | Mint/Green `#2EE6A8` 계열 | 강조, CTA, 로고 |
| Accent | Cyan Blue `#36B0F0` 계열 | 그라디언트, 링크 |
| Brand Gradient | Magenta → Cyan | 로고 심볼, 포인트 |
| Alert | Red `#FF4D4D` | The Problem 섹션 |
| Text | Off-white `#E8EEF5` / Muted Gray | 본문/보조 |

### 5.2 타이포그래피

- 영문 헤드라인: 굵은 산세리프 (Geometric Sans, 예: Poppins/Montserrat 계열)
- 국문 본문: 가독성 높은 산세리프 (Pretendard 권장)
- 로고 워드마크: 소문자 `GCM`, ü 디테일 강조

### 5.3 무드 & 모션

- 노출 콘크리트/메탈/네온 질감의 배경 텍스처
- 스크롤 기반 페이드/패럴럭스, 다크 배경 위 네온 글로우
- 절제된 마이크로 인터랙션 (프리미엄 톤 유지, 과한 애니메이션 지양)

## 6. 기술 스택 제안

| 영역 | 제안 | 이유 |
|------|------|------|
| Framework | Next.js (App Router) | SEO, 정적/동적 하이브리드, 스크롤 내러티브 적합 |
| 언어 | TypeScript | 타입 안정성 |
| 스타일 | Tailwind CSS + 디자인 토큰 | 다크 테마/그라디언트 일관성 |
| 애니메이션 | Framer Motion | 스크롤 인터랙션 |
| CMS | Headless CMS (Sanity/Contentful) 또는 MDX | 프로그램/스토리/공지 운영 |
| 폼/상담 | 서버리스 + 이메일/Slack 알림 | 상담·B2B 문의 |
| 배포 | Vercel | Next.js 최적화, 미리보기 배포 |
| 분석 | GA4 / 전환 추적 | 상담 전환 측정 |

> 콘텐츠 위주 마케팅 사이트이므로 별도 백엔드 없이 시작 가능.
> 상담 예약·회원/크루 관리가 본격화되면 BaaS(인증/DB) 또는 전용 API 도입 검토.
